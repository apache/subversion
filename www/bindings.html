Subversion API bindings

The library-based architecture of Subversion's canonical C language implementation presents an easily hackable API around which to create bindings for other languages. Existing bindings generally cover the Subversion client API.

  • Python
  • Java
  • Perl
  • Ruby

"Version control with Subversion" has some coverage of bindings, and a list of full blown client applications.
